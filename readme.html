System requirements

This program will run on any system with the following requirements:
  • Perl 5.004+
  • Time::* Perl modules, which you can obtain from the /modules/by-module/Time directory on CPAN. The file name should look something like Time-modules-98.060902.tar.gz, but get the latest version available. If you are running ActivePerl, you can just type "install time-modules" from the ppm
  • An HTTP (Web) server which can run CGI programs
  • A text editor (you will need to set some values for your local configuration)
  • A cron process of some variety to run the expire_reservations script - or you can run it manually every once in a while, if you prefer
